Neural Causal Regularization (ncr)

ncr is a lightweight Python library that implements the ℓ1 risk‑gap penalty proposed in Neural Causal Regularization: Extending Causal Invariance to Deep Models (Richter et al., 2025). It helps neural networks ignore spurious correlations and generalise to unseen environments.

Highlights

  • Two‑environment objective – no need for many domains.
  • Works out‑of‑the‑box with any PyTorch model.
  • Grid‑search helper to auto‑tune the penalty weight λ.
  • Clean API – only four public functions.

Quick install

pip install torch   # if you don't have it
pip install ncr
