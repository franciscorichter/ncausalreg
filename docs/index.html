Neural Causal Regularization Documentation

Welcome to the documentation for the Neural Causal Regularization (NCR) package. This documentation provides comprehensive information about the package, including installation instructions, theoretical background, API reference, and examples.

What is Neural Causal Regularization?

Neural Causal Regularization is a framework that extends causal invariance principles to deep neural networks. It addresses the challenge of distribution shifts in machine learning by encouraging models to learn stable, invariant relationships that generalize better to new, unseen environments.

The key idea behind NCR is to penalize the variance of prediction risks across different environments during training. This encourages the model to focus on features that lead to consistent performance across environments, which typically correspond to causal relationships.

Key Concepts

Causal Invariance

Causal invariance is based on the principle that causal relationships remain stable across different environments or interventions. By focusing on learning these invariant relationships, models can generalize better to new environments.

Out-of-Distribution Generalization

Out-of-distribution (OOD) generalization refers to a model's ability to perform well on data that comes from a different distribution than the training data. This is a critical challenge in many real-world applications where test conditions may differ from training conditions.

Structural Equation Models

Structural Equation Models (SEMs) provide a framework for representing causal relationships between variables. In the context of NCR, SEMs are used to generate synthetic data for benchmarking and to understand the theoretical properties of the method.

Package Components

Data Generation

The package provides functions to generate data from various structural equation models:

  • generate_linear_sem(): Generate data from linear structural equation models
  • generate_nonlinear_sem(): Generate data from nonlinear structural equation models
  • generate_hidden_confounders_sem(): Generate data with hidden confounders
  • generate_adversarial_sem(): Generate data with adversarial environment shifts

Model Training

Functions for creating and training models with causal regularization:

  • ncr_model(): Create a neural network model for causal regularization
  • compute_ncr_loss(): Compute the causal regularization loss
  • train_ncr(): Train a neural network with causal regularization
  • train_linear_causal_reg(): Train a linear model with causal regularization

Evaluation and Visualization

Functions for evaluating models and visualizing results:

  • evaluate_model(): Evaluate a model on test data
  • compare_models(): Compare multiple models on test data
  • extract_feature_importance(): Extract feature importance from a trained model
  • plot_training_history(): Plot training history
  • plot_model_comparison(): Plot model comparison results
  • plot_feature_importance(): Plot feature importance
