FAQ

General

What problem does NCR solve?

It reduces a model’s reliance on spurious features by equalising the prediction risk across (at least) two training environments.

How many environments do I need?

Two diverse environments are enough for the theory and for the coloured‑MNIST benchmark. More can help tune λ.

Is this the same as IRM or REx?

NCR is closest to REx but uses an ℓ1 gap instead of variance, which is convex, scale‑free, and needs only one hyper‑parameter.

Practical

How do I choose λ?

Call ncr.utils.tune_lambda with a validation loader from a second environment. The helper tries several values and returns the best.

Can I use CNNs?

Yes. Pass any torch.nn.Module to train_ncr.

GPU support?

The code is pure PyTorch; it runs on CPU or CUDA automatically.
