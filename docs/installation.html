Installation

This page provides instructions for installing the Neural Causal Regularization package.

Prerequisites

Before installing the Neural Causal Regularization package, you need to have R installed on your system. The package has been tested with R version 4.0.0 and above.

The package depends on the following R packages:

  • torch: For neural network implementation
  • ggplot2: For visualization
  • dplyr: For data manipulation
  • tidyr: For data reshaping

Installation from GitHub

The recommended way to install the Neural Causal Regularization package is from GitHub using the devtools package:

# Install devtools if not already installed
if (!requireNamespace("devtools", quietly = TRUE)) {
  install.packages("devtools")
}

# Install ncausalreg from GitHub
devtools::install_github("username/ncausalreg")

Installation from Source

Alternatively, you can download the package source from the Download page and install it locally:

# Install from local source
install.packages("path/to/ncausalreg_package.zip", repos = NULL, type = "source")

Installing Dependencies

The package dependencies should be installed automatically. However, if you encounter any issues, you can install them manually:

# Install dependencies
install.packages(c("ggplot2", "dplyr", "tidyr"))

# Install torch
install.packages("torch")

Note that torch is a large package and may take some time to install. It includes binaries for various platforms, so no additional configuration should be needed.

Verifying Installation

To verify that the package has been installed correctly, you can load it and check the version:

# Load the package
library(ncausalreg)

# Check package version
packageVersion("ncausalreg")

Troubleshooting

Common Issues

Torch Installation Issues

If you encounter issues with the torch installation, you can try installing it separately before installing ncausalreg:

# Install torch separately
install.packages("torch")

# Verify torch installation
library(torch)
torch_tensor(1)

Package Not Found

If R cannot find the package after installation, make sure that your R library path is correctly set:

# Check library paths
.libPaths()

# Install to a specific library path
devtools::install_github("username/ncausalreg", lib = .libPaths()[1])

Dependency Conflicts

If you encounter dependency conflicts, you may need to update your existing packages:

# Update packages
update.packages(ask = FALSE)

System Requirements

The Neural Causal Regularization package has been tested on the following systems:

  • Windows 10 and 11
  • macOS 10.15 (Catalina) and later
  • Ubuntu 20.04 LTS and later

For optimal performance, especially when training large neural networks, we recommend:

  • At least 8GB of RAM
  • A modern CPU with multiple cores
  • GPU support (optional, but can significantly speed up training)
