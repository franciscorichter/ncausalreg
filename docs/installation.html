Installation

Prerequisites

  • Python 3.9+
  • torch ≥ 2.0 (CPU or CUDA build)

PyPI (recommended)

pip install torch           # CPU build
pip install ncr             # installs pure‑Python wheel

Editable install (developers)

git clone https://github.com/your-org/ncr.git
cd ncr
python -m pip install -e .
pytest -q tests             # optional: run unit tests

GPU support

Install the CUDA wheel of PyTorch first, then pip install ncr. Nothing else changes.
