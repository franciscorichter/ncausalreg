Theoretical Background

Objective

Given two training environments e = 1,2 with empirical risks \(R_e(\theta)\), NCR minimises

 L(θ) = ½(R₁ + R₂)  +  γ · |R₁ − R₂|,  γ ≥ 0

or equivalently a convex combination \( (1−λ)·½(R₁+R₂) + λ·|R₁−R₂|\) with λ ∈ [0,1).

Interpretation

  • γ = 0 reproduces ERM.
  • γ→∞ forces R₁≈R₂, selecting predictors whose loss is invariant — typically causal.
  • The ℓ1 gap is stricter than the REx variance penalty \(½(R₁−R₂)²\).

Consistency sketch

Under a standard structural‑causal‑model setup (shift variable A affects spurious features of X but not Y given causal subset XS), minimising L(θ) converges to the oracle using only XS.

Relation to IRM / REx

  • IRM: bi‑level optimisation, harder to tune.
  • REx: variance penalty; NCR replaces the second‑moment with a sharper ℓ1 gap.
