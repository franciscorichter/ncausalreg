API Reference

ncr.model

NCRNet(input_dim, hidden_dims=(256,256), output_dim=1)

Fully connected network: input → hidden_dims → output_dim. Flattening happens automatically so it works on images or vectors.

ncr.train

train_ncr(model, env_loaders, criterion, *, penalty=risk_gap, epochs=100, lambda_reg=1.0, lr=1e‑3, device="cpu")

  • env_loaders: list of torch.utils.data.DataLoader, one per environment (≥ 2).
  • criterion: e.g. torch.nn.MSELoss or BCE.
  • penalty: risk_gap (default) or risk_variance.
  • lambda_reg: γ in the objective; 0 ⇒ ERM.

ncr.penalty

risk_gap(r1, r2)

Absolute difference |R₁ − R₂|.

risk_variance(r1, r2)

0.5 · (R₁ − R₂)² — used by REx.

ncr.utils

accuracy(model, loader, device="cpu")

Returns fraction correct for binary classification.

tune_lambda(model_ctor, env_loaders, val_loader, criterion, lambdas=(0.1,1,10), …)

Grid‑searches λ; returns (best_model, best_lambda, history).
