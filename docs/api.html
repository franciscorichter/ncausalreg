Python API

Below is the complete public surface of the ncr package.

1 · Core training

train_ncr

train_ncr(model, env_loaders, criterion,
         *, penalty=risk_gap, lambda_reg=1.0,
         epochs=100, lr=1e-3, device="cpu")
  • env_loaders – list of DataLoader; each defines an environment.
  • penalty – risk_gap (ℓ₁) or risk_variance (REx).
  • Set lambda_reg = 0 for vanilla ERM.

NCRNet

NCRNet(input_dim:int,
       hidden_dims:tuple[int,...]=(256,256),
       output_dim:int=1)

2 · Penalties

risk_gap(r1, r2)        # |R1 - R2|
risk_variance(r1, r2)   # 0.5 * (R1 - R2)**2

3 · Synthetic data

generate_linear_sem(...)
generate_nonlinear_sem(...)

Return dicts with X, Y, A, env. More generators coming soon.

4 · Linear baseline

train_linear_causal_reg(x_train_list, y_train_list,
                        lambda_reg=1.0)

5 · Evaluation helpers

accuracy(model, loader[, device])
evaluate_model(model, x_test, y_test)
compare_models(models, names, x_test_list, y_test_list)
tune_lambda(model_ctor, env_loaders, val_loader, ...)

6 · Visualisation

plot_training_history(training_result)
plot_feature_importance(importance_df[, top_n])

Return a matplotlib Axes; customise as needed.
